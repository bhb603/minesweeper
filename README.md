# Minesweeper

Play minesweeper in the terminal!

![demo](./docs/demo.gif)


## Install

Go:
```
go install github.com/bhb603/minesweeper/cmd/minesweeper-cli@latest
```

Docker:
```
docker pull ghcr.io/bhb603/minesweeper
```

## Usage

```
minesweeper-cli
```

Docker:
```
docker run --rm -it ghcr.io/bhb603/minesweeper-cli
```
